Steps to set up Git for Linux

Install Git

  1. Update the system: sudo pacman -Syuz
  2. Install git: sudo pacman -S git
  3. Verify version: git --version

Configure Git

  1. Set up user name: git config -global user.name "Your Name"
  2. Set up user's email: git config --global user.email "yourmail@example.com"

Set up GitHub

  1. Create a GitHub account or sign in if you already have one.
  2. Create an SSH key for your local machine: ssh-keygen -t ed25519 -C yourmail@example.com
  3. Link your SSH Key with GitHub.
  4. Test the SSH key: ssh -T git@github.com
